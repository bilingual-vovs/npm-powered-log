# CusomTypes 
## **`Dir`**
> ### **Syntax**
> 
>       new Dir(someDir)
> 
> **Where**
> * `someDir` - direction to create
> **Output**
> object - 
>   * dir - direction *(string)*

## **`Color`**
> ### **Syntax**
> 
>      new Color(someColor)
> 
> **Where**
> * `someColor` - color to create
> **Output**
> object - 
>   * color - color *(string)*
